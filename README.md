## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/funnypar/profile_app.git
```

### Go to Project Folder

```bash
cd profile_app
```

### Install Dependencies

```bash
flutter pub get
```

### Run App

```bash
flutter run
```
